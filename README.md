# NISROOM

## Build Setup

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start
```

## Create `.env` File

```.env
API_URL=
API_KEY=
```

### 1. Fork and Clone

### 2. `git remote add upstream https://github.com/Kenny-NISLab/nisplan.git`
